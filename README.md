taiwan.py
=========

台灣的Python社群

# 安裝環境:
1. `sudo pip install mynt`
2. `git clone https://github.com/lucemia/taiwan.py.git`
3. `cd taiwan.py`
4. `mynt gen -f src/ build/`
5. `mynt watch -f src/ build/`
6. `mynt serve build/`
